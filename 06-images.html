Notes:

  • We have added "border:0" to prevent IE9 (and earlier) from displaying a border around the image (when the image is a link).
  • The alt attribute is required for web page validation
  • Always specify the width and height of an image. ( the page might flicker while the image loads )
  • we suggest using the style attribute: prevents styles sheets from changing the size of images.
