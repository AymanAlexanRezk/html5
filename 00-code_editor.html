VSCode Editor EMMET Tips

EMMET Abbr. & some VS Code ( How To )

  • EMMET : Tab Or Enter
  • For Basic Layout: ! +Enter or Tab
  • Double click a file to be opened & maintained
  • On the selected file: +add new file
  • Comment : (CTRL + ?/)
  • CTRL + Enter ( to begin a new line without breaking the line you're in )
  • lorem + Tab for dummy text
  • lorem10 + Tab: for 10 dummy words
  • SHIFT + ALT + Down => to copy the line you're in in a new line
  • CTRL + left/right arrow to move by word ( not by a singl space )
  • Press Home key to move to the begining of the line
  • Press End key to move to the end of the line
  • Alt + up or down arrow to move the line you're in
  • li*4 + Tab to have four list items
  • input:email +Tab
  • select a word , then: ctrl + d to select the other one and so on
  • Select multi-lines then hit Tab to indent them more to the right,
    Or: Shift + Tab to indent them back
  • div#about +Tab || #about +Tab == a div of id="about"
  • h1#about +Tab == h1 element with id="about"

VSCode Extensions

  1. Live Server => generate a loopback address:

    An address that sends outgoing signals back to the same computer for testing.
    In a TCP/IP network, the loopback IP address is 127.0.0.1,
    and pinging this address will always return a reply
    unless the firewall prevents it.

    The loopback address allows a network administrator to treat the local machine
    as if it were a remote machine.
